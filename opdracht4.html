Opdracht 4

Itereren over opties

Haal alle opties uit de select met de id "color" en sla deze op in een variabele. Doe hetzelfde voor de select met de id "vehicle". Je hebt nu twee arrays. Maak een for loop die itereert over de array met kleuren. (Tip: Gebruik de lengte van de array als maximale waarde). En schrijf binnen deze for loop nog een for loop die itereert over alle voertuigen. Gebruik deze for loops om voor elke mogelijke combinatie van kleur en voertuig een alert te tonen. De geselecteerde opties zijn niet belangrijk. Het gaat in deze opdracht om alle mogelijke combinaties.

Voor gevorderde

Gebruik een foreach loop om dit probleem op te lossen.
